# 星枢令 (ArcaneKey) — 鸿蒙端功能对齐

## What This Is

**星枢令**是一款跨平台 TOTP 身份验证器。微信小程序端已完成 v1.0 + v1.1 验证，本项目将已验证的小程序功能逻辑同步到鸿蒙 App，实现两端功能对齐。

**核心价值：** 让鸿蒙用户获得与微信小程序用户完全一致的 TOTP 身份验证体验。

---

## Context

- **平台：** HarmonyOS 6.1（API 23），ArkTS
- **当前分支：** `dev-harmonyos`
- **参考基准：** 微信小程序 `main` 分支（已验证 v1.0 全功能 + v1.1 分享）
- **技术栈：** ArkTS、@kit.ScanKit、@kit.ArkData、AppGallery Connect（目标）
- **Bundle ID：** 当前 `com.example.authenticator`（需修正）
