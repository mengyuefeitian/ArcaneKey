在 HarmonyOS 6 (API 23) 中，底部悬浮菜单与沉浸光感是两个紧密关联的视觉与交互特性，旨在打造更具层次感和现代感的用户体验。

* **沉浸光感 (Immersive Light Effects)**：这是一个系统级的视觉增强系统。它能为锁屏、桌面、通知中心等界面带来半透明的毛玻璃（玻璃拟态）效果和动态光影，支持强、均衡、弱三档效果调节，让界面更具通透感和呼吸感。
* **底部悬浮菜单 (Float Navigation)**：这是一种全新的底部导航交互模式。导航栏不再紧贴屏幕底部，而是以悬浮的形式呈现，不占用布局空间。它能根据用户的滑动操作智能地隐藏或显示，从而最大化内容显示区域，提升沉浸感。

下面将通过一个“沉浸式音乐播放器”的开发案例，为你详细解析如何实现这两个特性。

### 🎵 实战案例：打造沉浸式音乐播放器

这个案例将展示如何构建一个具有悬浮导航和沉浸光感效果的音乐播放器应用。

#### 1. 项目配置与权限声明

首先，需要在项目的 `module.json5` 配置文件中声明必要的权限，特别是悬浮窗权限，这是实现悬浮效果的基础。

```json5
// module.json5
{
  "module": {
    "name": "entry",
    "type": "entry",
    // ...其他配置
    "requestPermissions": [
      {
        "name": "ohos.permission.SYSTEM_FLOAT_WINDOW",
        "reason": "$string:float_window_permission",
        "usedScene": {
          "abilities": ["EntryAbility"],
          "when": "always"
        }
      },
      {
        "name": "ohos.permission.INTERNET",
        "reason": "$string:internet_permission"
      }
    ]
  }
}
```

#### 2. 配置窗口沉浸模式

在应用的入口 `EntryAbility.ets` 中，配置窗口的全屏和背景属性，这是实现沉浸光感的关键一步。通过设置窗口背景为透明，允许系统的光效穿透。

```typescript
// entry/src/main/ets/entryability/EntryAbility.ets
import { window } from '@kit.ArkUI';
import { BusinessError } from '@kit.BasicServicesKit';

export default class EntryAbility extends UIAbility {
  private windowStage: window.WindowStage | null = null;

  onWindowStageCreate(windowStage: window.WindowStage): void {
    this.windowStage = windowStage;
    // 加载主页面
    windowStage.loadContent('pages/Index', (err) => {
      if (err.code) {
        console.error('Failed to load content:', JSON.stringify(err));
        return;
      }
      // 配置沉浸窗口
      this.setupImmersiveWindow(windowStage);
    });
  }

  private async setupImmersiveWindow(windowStage: window.WindowStage): Promise<void> {
    try {
      const mainWindow = windowStage.getMainWindowSync();
      // 1. 设置窗口全屏布局，内容延伸至状态栏和导航栏
      await mainWindow.setWindowLayoutFullScreen(true);
      // 2. 设置窗口背景为透明，允许光效穿透
      await mainWindow.setWindowBackgroundColor('#00000000');
      // 3. 配置系统栏属性，如透明背景等
      // ...
    } catch (err) {
      console.error('Failed to setup immersive window:', JSON.stringify(err));
    }
  }
}
```

#### 3. 构建悬浮导航与玻璃拟态UI

在主页面 `Index.ets` 中，我们将使用 `Stack` 布局来构建页面，将内容区域和悬浮导航栏分层放置。同时，利用 `backdropFilter` 属性实现玻璃拟态效果。

```typescript
// entry/src/main/ets/pages/Index.ets
@Entry
@Component
struct Index {
  @State currentIndex: number = 0;
  // 模拟沉浸光感的强度，实际开发中可从系统获取
  @State lightIntensity: number = 0.6; 

  build() {
    Stack() {
      // 1. 主要内容区域
      // 这里放置你的音乐播放器主界面，如专辑封面、播放列表等
      Column() {
        Text('音乐播放器主内容')
          .fontSize(24)
          .margin({ top: 50 })
      }
      .width('100%')
      .height('100%')
      // 关键：让内容区域自动适配底部悬浮导航的高度，避免遮挡
      .padding({ bottom: 80 }) 

      // 2. 底部悬浮导航栏
      Row({ space: 20 })
        .position({ x: 0, y: '100%' }) // 定位在底部
        .translate({ y: -20 }) // 向上偏移，形成悬浮效果
        .justifyContent(FlexAlign.SpaceAround)
        .alignItems(VerticalAlign.Center)
        .width('90%')
        .height(60)
        .margin({ bottom: 20 })
        .borderRadius(30)
        // 实现玻璃拟态效果
        .backgroundColor(`rgba(30, 30, 40, ${0.3 + this.lightIntensity * 0.3})`)
        .backdropFilter(`blur(${10 + this.lightIntensity * 10}px)`)
        .shadow({ radius: 10, color: 'rgba(0,0,0,0.2)', offsetX: 0, offsetY: 5 })
      
        // 导航项
        ForEach(['首页', '发现', '我的'], (item: string, index: number) => {
          Column() {
            Text(item)
              .fontSize(14)
              .fontColor(this.currentIndex === index ? '#4ECDC4' : '#FFFFFF')
          }
          .onClick(() => {
            this.currentIndex = index;
          })
        })
    }
    .width('100%')
    .height('100%')
  }
}
```

### ✨ 核心要点总结

* **权限是前提**：实现悬浮效果必须先在 `module.json5` 中申请 `ohos.permission.SYSTEM_FLOAT_WINDOW` 权限。
* **窗口配置是基础**：在 `EntryAbility` 中通过 `setWindowLayoutFullScreen(true)` 和 `setWindowBackgroundColor('#00000000')` 开启窗口的沉浸模式。
* **布局是关键**：使用 `Stack` 或 `Float` 布局将导航栏与内容分离，并通过 `position` 和 `translate` 属性实现悬浮的视觉效果。
* **玻璃拟态靠滤镜**：使用 `.backdropFilter('blur(...)')` 属性配合半透明背景色，是实现毛玻璃质感的核心。
* **智能交互**：在实际应用中，可以监听页面的滚动事件，动态控制底部导航栏的显示与隐藏，实现更智能的交互体验。
