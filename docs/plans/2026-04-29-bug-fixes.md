# Bug Fix Plan — 2026-04-29

## Bug 1: 底部导航点击无效

**文件**: `app.jsx` — `BottomNav` 组件 (line 242-287)

**分析**: 底部导航按钮可能缺少关键样式属性。修复:
- 给按钮显式添加 `type="button"` 防止默认提交行为
- 添加 `userSelect: 'none'` 防止文字选中
- 外层容器增加 `minHeight` 确保按钮可点击区域足够大
- 确保按钮 `lineHeight` 正常

## Bug 2: 编辑账号右上角多余保存按钮

**文件**: `app.jsx` — `EditModal` 组件 (line 585-640)

**方案**: 移除 header 中的保存按钮,保留标题 + 关闭按钮。将保存按钮移到页面底部,与删除按钮同区域,上下排列。

## Bug 3: 搜索框太占空间且无法关闭

**文件**: `app.jsx` — `HomeScreen` 组件 (line 307-338)

**方案**: 搜索激活时,header 同时显示标题和搜索框(不再互相替换)。搜索框缩成一行小输入框(高度 ~36px),右侧带 X 关闭按钮。标题保留不消失。
